Installing JBoss Tools 4.1.0

Requirements | What's New | Release Note
Try latest developments | Archived versions
JBoss Tools 4.1.0 was released on 2013-07-22. Read the announcement on our blog to know more about it.
  • Eclipse Marketplace
  • Update Site
  • Downloads

Drag and drop the icon on your active Eclipse Kepler workspace, or alternatively, search for "JBoss Tools (Kepler)" in Eclipse via:

Help > Eclipse Marketplace...

You can also view the installation page on the Eclipse Marketplace.

Add the following URL to your Eclipse Kepler workspace: http://download.jboss.org/jbosstools/updates/stable/kepler/ via

Help > Install New Software...

then select the individual features that you want to install.

You can download Update Zips (Archived p2 Repos) for offline installation of JBoss Tools, but DO NOT UNPACK THEM into your Eclipse root or dropins folder! Next, install into Eclipse using

Help > Install New Software... > Add... > Archive...

and point to the archive you downloaded from the tables below, depending on the features of JBoss Tools you want to use.

Name Size Links
Update site (including sources) bundle of all JBoss Tools Core 409MB zip (md5)
Sources zip of all JBoss Tools Core 135MB zip (md5)
Update site (including sources) bundle of all Hibernate Tools 78MB zip (md5)

JBoss Tools 4.1.0 Requirements

JBoss Tools 4.1.0 runs on Windows, MacOSX and Linux and requires Eclipse Kepler
