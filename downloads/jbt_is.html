JBoss Tools Integration Stack 3.3.3.Final was released on 2013-02-19.

Add the following URL to your Eclipse Indigo 3.7.2 workspace: http://download.jboss.org/jbosstools/updates/stable/indigo/soa-tooling/ via

Help > Install New Software...

then select the individual features that you want to install.

You can download Update Zips (Archived p2 Repos) for offline installation of JBoss Tools Integration Stack, but DO NOT UNPACK THEM into your Eclipse root or dropins folder! Next, install into Eclipse using

Help > Install New Software... > Add... > Archive...

and point to the archive you downloaded from the tables below, depending on the features of JBoss Tools you want to use.

Name Size Links
Update site (including sources) bundle of all JBoss Tools SOA 87MB zip (md5)
Sources zip of all JBoss Tools SOA 854KB zip (md5)
